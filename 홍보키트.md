# 뽑뽑 홍보 키트

> 순서: **제보 20~30개 채우기 → 콘텐츠 만들기 → 커뮤니티 알리기**
> 빈 지도로 홍보하면 첫인상을 되돌릴 수 없습니다. 반드시 채우고 시작하세요.

---

## 0. 시작 전 체크리스트

- [ ] 지역 하나 선정 (홍대 / 강남 / 건대 등 뽑기방 밀집 지역)
- [ ] 그 지역 제보 **20곳 이상**, 상품 **40개 이상**
- [ ] 다니면서 사진·영상 확보 (릴스용 세로 영상)
- [ ] 앱 링크 준비: `https://bbopbbop-s3hg.vercel.app`
- [ ] 첫 화면 직접 확인 — 처음 들어온 사람이 뭘 보는지

> **중요:** 전국에 흩뿌리지 말고 한 지역에 몰아넣으세요.
> 넓이보다 밀도입니다. 한 동네에서 "여기선 진짜 쓸만하다"가 되어야 번집니다.

---

## 1. 인스타그램

### 릴스 (가장 효과 큼)

**영상 구성 (15~30초)**
1. 훅: "홍대 뽑기방 20곳 다 돌아봤습니다" (자막 크게)
2. 빠른 컷: 가게 외관 → 인형들 → 뽑는 장면
3. 마무리: 앱 지도 화면 보여주며 "다 정리해놨어요"

**캡션**
```
홍대 뽑기방 20곳 다 돌면서 뭐가 있는지 전부 정리했습니다 🧸

친구들이 자꾸 "홍대 어디가 잘 나와?" 물어봐서
아예 지도에 다 박아놨어요.

어디에 뭐가 있는지, 언제 확인된 정보인지까지
프로필 링크에서 볼 수 있어요.

다음은 강남 갑니다. 원하는 동네 댓글로 남겨주세요

#인형뽑기 #뽑기방 #홍대뽑기방 #인형뽑기성지 #홍대놀거리
#크레인게임 #인형뽑기추천 #홍대데이트 #뽑기 #인형
```

### 피드 (지도 스크린샷)

```
🧸 뽑뽑 — 어디서 뭘 뽑지?

내 주변 인형뽑기방에 지금 뭐가 있는지
사람들이 직접 올리는 지도예요.

· 어디에 무슨 인형이 있는지
· 언제 확인된 정보인지
· 안 쓰는 인형 중고거래까지

제보하면 포인트도 쌓여요.
프로필 링크에서 바로 쓸 수 있어요 (설치 없이)

#인형뽑기 #뽑기방 #인형뽑기성지
```

**팁**
- 스토리에 "우리 동네도 해줄까요?" 투표 올리면 반응 확인 + 다음 지역 결정 동시에 됨
- 프로필 링크 필수. 게시물 안 링크는 안 눌림

```

혼자 다 채우기는 무리라
다니시는 분들이 하나씩 올려주시면 좋을듯
```

- 연속 도배 ❌

## 3. 네이버 블로그 / 카페

> 검색 유입을 노리는 채널. 인스타·디시와 달리 **오래 남습니다.**

### 블로그 글 제목 (검색어 겨냥)

- `홍대 인형뽑기방 추천 20곳 총정리 (2026년 최신)`
- `홍대 뽑기방 어디가 잘 나올까? 직접 다 돌아봤습니다`

### 구성

```
1. 도입 — 왜 정리하게 됐는지 (친구가 자꾸 물어봐서 등)
2. 지역별 가게 목록
   - 가게명 / 위치 / 사진 / 어떤 인형이 있는지 / 체감 난이도
3. 마무리 — "실시간으로 바뀌니까 지도로 만들었어요" + 링크
4. 지도 스크린샷 첨부
```

**팁**
- 사진 많이, 글은 실제 경험처럼
- 제목에 지역명 + "인형뽑기" 조합이 검색에 걸립니다
- 카페는 각 카페 규칙 확인 필수 (홍보 금지인 곳 많음)

## 6. 반응 체크 지표

홍보 후 이것만 보세요. 나머지는 노이즈입니다.

- **신규 가입 수** — 링크가 실제로 눌렸는가
- **제보 올린 사람 수** — 그냥 구경이 아니라 참여했는가 ← 가장 중요
- **다음날 재방문** — 한 번 쓰고 마는지, 다시 오는지

가입은 느는데 제보가 안 늘면 **참여 유도**에 문제가 있는 겁니다.
방문이 한 번뿐이면 **콘텐츠 밀도**가 부족한 거고요.
